Detailed Description

Sensor access for an ICM20649 IMU Sensor. Child of the abstract IMUSensor class.

Note
supports I2C and SPI communication.
See also
Adafruit library this class utilizes: https://adafruit.github.io/Adafruit_ICM20X/html/class_adafruit___i_c_m20_x.html
