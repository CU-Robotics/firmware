Detailed Description

MG8016E-i6v3 controller driver. This manages generating CAN output messages and processing incomming CAN messages into a state array.

Motor datasheet: http://en.lkmotor.cn/upload/20230321150047f.pdf

CAN datasheet: http://en.lkmotor.cn/upload/20230706100134f.pdf

Note
Motors must be explicitly constructed to avoid uninitialized parameters. Motors exist to be managed by CANManager.
