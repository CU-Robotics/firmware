Detailed Description

Motor driver for the GIM motor.

Note
GIM8108 docs (SDC103): https://steadywin.cn/en/pd.jsp?id=15&fromColId=0#_pp=0_752_3
GIM4310 docs : https://steadywin.cn/en/pd.jsp?id=11&fromColId=0#_pp=0_752_3
GIM3505 docs : https://steadywin.cn/en/pd.jsp?id=130&fromColId=0#_pp=0_752_3
motor driver protocol: https://14180476.s21i.faiusr.com/61/ABUIABA9GAAgqPvVqgYo-uLA7Ac.pdf?v=1708485774

Common write command. This fills the given message if successful. This is used to compile data to be sent over the CAN line.

Parameters
msgThe message buffer to fill data into
Returns
The index in the buffer where the motor data was written. The MG8016E-i6 driver will always write 8 bytes so this will always return 0
Note
Does not issue a CAN command over the bus
Return value is mostly useless for this motor, it will always be 0
